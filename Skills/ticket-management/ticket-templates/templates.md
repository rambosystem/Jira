# Team Ticket Templates

Use these templates when creating issues for the CP team.

## Story Template

```markdown
## User Story
As a [role], I want [capability], so that [benefit].

## Business Value
[Why this matters]

## Scope
- Component: [component]
- In scope: [items]
- Out of scope: [items]

## Acceptance Criteria
- [ ] Scenario 1
- [ ] Scenario 2
- [ ] Non-functional constraints are met

## Notes / Links
- Jira/Doc/PR links
```

## Epic Template

```markdown
## Background
[Why this quarter-level epic is needed]

## Goal
[What this epic should deliver by quarter end]

## Scope
- Component: [component]
- Delivery Quarter: [Q1/Q2/Q3/Q4]
- In scope: [major capabilities]
- Out of scope: [excluded items]

## Acceptance Criteria
- [ ] Epic structure and scope are confirmed
- [ ] Delivery Quarter is set correctly
- [ ] Roadmap label is set correctly
- [ ] Ready for child story breakdown

## Notes / Links
- Jira/Doc/PR links
```
